PKTraffic LLC

×

PK Traffic specializes in deep public record search of traffic related data and events to provide expert analysis and support. The business model of the company is to provide insights in traffic incidents through post-processing to support safety and justice to traffic crash victims. Traffic related incidents is analyzed through an intelligent application which makes it easy to explore, analyze and visualize.

Casper Kristiansson is one of the owners and CTO (chief technology officer) of the company. He developed both the frontend and backend parts of the IT Solutions enabling the business model. The frontend part of includes a program which is able to visualize data and sort through traffic crashes, and the website pktraffic.com. The backend of the project includes managing a server which handles a large amount of data and camera footage. His main responsibility is to assure well working IT solutions, that the data is secure and of good quality.

Website

TvShowNetwork

×

TvShowNetwork is an article publishing website where a user is able to sign up and write their own reviews/articles on film and the TV industry. Starting a new blog can be hard, especially with competing against sites that posts 5-10 articles a day. Climbing the google search ladder is harder than ever. By writers joining together they can quickly beat the bigger article websites. The objective of the project is that the writers receive a share of what their article websites have received in google AdSense. During the first year of the website launching, it had attracted approximately 166,000 visitors.

Casper Kristiansson has developed the business model, the website and the backend of the site which includes keeping track on which and how many visitors each article author gets. The project also includes optimizing the website for SEO (search engine optimization).

Image Editor

×

Casper Kristiansson developed a mass image editing program for the Personnel Foundation of the Swedish postal service (postnordplus.com). The objective of the program is to mass edit images which only requires a few changes such as changing the width and height, brightness, sharpness, vignette, vibrance and much more. The program applies all changes to the selected images. The program also included a version where the user is able to go through an image at a time a changing the image properties. The program enables the PostNordPlus to save large amount of time specifically for images that will be uploaded to a website.

Casper Kristiansson was responsible to develop both the frontend and backend of the program which includes a functionality to manipulate and work with images. He was also responsible for training of the staff in using the program and is on request supporting the end users when needed.

Twitch YouTube Auto Uploader

×

The purpose of the project is to automatically upload twitch clips to YouTube using both Twitch API and googles API. On the website twitch a user is able to “clip” a part of a livestream and upload it to the platform. These clips often get a lot of attention and often attracts millions of views. With this solution using twitch API and YouTube dl makes it possible to download popular clips from the platform and uploading the clips to YouTube using their data v3 API.

Automated Instagram Account
